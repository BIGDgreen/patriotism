首页
首页默认情况下展示全部文章、视频（可以上拉刷新、下拉加载和回到顶部）
搜索包含输入搜索和标签搜索，点击分类，可以进入“标签搜索”
点击文章、视频列表中的一项，可以进入详情页

用户选择的文章/视频详情，点击左上角返回按钮，可以返回到首页；点击右上角分享按钮，可以将该页面分享到其他平台
可以在最下方输入框中输入评论并发表
